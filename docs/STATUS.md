## 一、呜哩是什么

呜哩是一个**面向 3-8 岁儿童的人工智能成长伙伴**。

孩子通过语音或文字和一个叫"呜哩"的外星小朋友聊天。聊天过程中，系统会自动评估孩子在四个维度上的能力发展：

| 能力 | 含义 |
|------|------|
| 创造力 | 想象力、发散思维、能不能想出不一样的点子 |
| 批判思维 | 会不会问"为什么"、能不能把事情讲出因果关系 |
| 沟通表达 | 能不能把自己的想法清楚地说出来 |
| 协作力 | 能不能倾听、回应、配合对方 |

家长不需要陪在旁边。聊完之后，家长打开**家长端**就能看到：
- 孩子各项能力的分数和变化趋势
- 孩子聊天中聊到了什么（兴趣、朋友、经历）
- 达成了哪些"里程碑"（比如第一次在对话中使用了因果句式）
- 一封 AI 写的成长周报，用温柔的语言总结这周的观察

**一句话**：孩子跟呜哩聊天 = 家长收到一份专业的成长观察报告。

## 四、还缺什么

### 🔴 上线前必须做

| 项目 | 说明 | 预估工作量 |
|------|------|-----------|
| **部署上线** | 目前只跑在本地电脑，需要部署到服务器（域名 + HTTPS） | 需要 1-2 天 |
| **API Key 轮换** | 开发用的 AI 接口密钥需要更换为正式的 | 10 分钟 |
| **内容安全审核** | 确保 AI 不会生成不适合儿童的内容（需要加 output filter） | 需要 1-2 天 |
| **真人测试** | 找 3-5 个真实家庭测试，收集反馈 | 需要协调 |

### 🟠 强烈建议做

| 项目 | 说明 |
|------|------|
| **LLM 流式输出** | 目前呜哩回复要等 5-15 秒才能看到全部内容，改成逐字显示会体验好很多 |
| **讯飞语音替代** | 目前语音识别依赖 Chrome 浏览器，换成讯飞后所有浏览器都能用 |
| **数据库备份** | 目前数据存在一个 SQLite 文件里，需要定期备份机制 |
| **favicon 和品牌标识** | 浏览器标签页目前没有图标 |
| **运营后台** | 查看注册用户数、活跃度等运营数据 |

### 🟡 后续可以迭代

| 项目 | 说明 |
|------|------|
| 多孩子管理（添加 / 切换） | 目前注册时只能填一个孩子 |
| 家长推送通知 | 每次聊天结束后给家长发微信 / 短信通知 |
| 成长报告 PDF 导出 | 把周报导出为可以分享给爷爷奶奶的 PDF |
| 更多聊天场景 | 比如讲故事模式、问答模式、每日挑战 |
| 社区 / 排行榜 | 激励孩子持续使用 |
| iOS / Android 应用 | 目前只有网页版 |
