## 四、uli-brain 主题框架（多源合并 × 中国适配）

### 设计原则

1. **以中国指南为骨架**：5 大领域 → 14 子领域 → 32 目标
2. **用 Core Knowledge 填充知识内容**：每个子领域下的具体知识点
3. **美国特有主题替换为中国对应内容**
4. **对话场景优先**：选择适合在 AI 对话中展开的主题

## 七、建议的审核流程

1. **专家审核**：找 1-2 位幼教老师审核 50 个话题的适龄性和完整性（预计 2-4 小时）
2. **家长调研**：向 10-20 位家长发问卷，确认话题覆盖孩子日常兴趣
3. **上线后迭代**：收集实际对话中高频话题，补充遗漏的主题
4. **每季度更新**：根据季节、节日、热点话题动态调整
